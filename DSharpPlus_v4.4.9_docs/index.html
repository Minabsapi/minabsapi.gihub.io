DSharpPlus

DSharpPlus Documentation

DSharpPlus (D#+) is an unofficial .NET wrapper for the Discord API which was originally a fork of DiscordSharp. The library has since been rewritten to fit quality and API standards as well as target modern .NET implementations.

Getting Started

New users will want to take a look through the articles for quick start guides, tutorials, and examples of use. Once you've gotten through the articles, head on over to the API Documentation for all classes and methods provided by this library.
